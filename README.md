# verilog_alu
A 16-bit ALU designed in the Verilog HDL

## To run:
`verilog_alu>iverilog -o alu_obj alu.v`
followed by
`vvp alu_obj`
## OR
### CMD/Powershell (Windows)
`verilog_alu>iverilog -o alu_obj alu.v && vvp alu_obj`
### bash/zsh (Linux/macOS)
`verilog_alu>iverilog -o alu_obj alu.v ; vvp alu_obj`
